Access the live website here: https://gwy15.github.io/263-ux-case-study/
Or run these files with a simple local web server (ex. python -m http.server)


Basic HTML content used:

Images + alt text (img, alt)
Headers (h1, h2, h5)
Paragraphs (p)
Divs (div)
Links (a)
Strong text (strong)
Unordered lists and list items (ul, li)



Basic CSS Styling:

Modified padding and margins (padding-left/padding-right, margin-left/margin-right/margin-bottom)
Modified background and text colors (color, sectionsColor (fullPage.js))



Advanced Features:

Full screen scrolling with right-hand side navigation bar (via fullPage.js).
Horizontally scrolling bootstrap card views on smaller screens. 
Text size and formatting changes in response to devices/window sizes (@media, bootstrap col sizes, etc).



Plugins/Libraries:

https://alvarotrigo.com/fullPage/
JavaScript library allowing for the addition of scrollable full screen pages and navigation bar.

https://lesscss.org/
CSS extension library.

https://getbootstrap.com/
JavaScript/SASS toolkit enabling responsive web design.
